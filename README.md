# Controle de Territórios (iOS / SwiftUI)

Aplicativo iOS para gerenciamento e controle de territórios de pregação desenvolvido em SwiftUI e SwiftData.

## Funcionalidades
- Cadastro e listagem de territórios.
- Organização de cartões por número e status (Disponível, Trabalhando, Concluído).
- Registro de atribuições por publicador.

## Como compilar no Codemagic
1. Faça o upload deste projeto (.zip extraído) para o seu repositório no GitHub.
2. Conecte o repositório ao Codemagic.
3. Configure a assinatura digital (Apple API Key / Code Signing).
4. Inicie o Build para iOS para gerar o arquivo `.ipa`.
